• git clone path to repository
  • git status статус текущего репозитория
  • git add path to file - добавить новый файл под контроль ГИТа
  • git commit -am "added new file index.html" - фиксация изменений и дать им комментарий
  • git push - отправить на сервер(гитхаб) все закоммиченые изменения
  • git pull - забрать с сервера свежие изменения
  • git add. - добавит все новые файлы из текущей папки
